# Backend - Sistema de Rastreamento de Encomendas

Este projeto é uma **API RESTful** desenvolvida com **Node.js**, **Express** e **MySQL**, 
para gerenciar usuários, rastreamentos, histórico de entregas, notificações automáticas e feedbacks.

## Tecnologias
- Node.js
- Express
- MySQL2
- Bcrypt
- Dotenv
- Nodemon

## Instalação
```bash
npm install
npm run dev
